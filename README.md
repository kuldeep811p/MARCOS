# MARCOS (v2.0) — Multimodal AI Research & Creative OS

**marcos** is an intelligent, context-aware AI assistant and research operating system powered by the **Google Gemini API**. It provides a centralized, friction-free workspace where users can ask complex questions, analyze multi-format data, extract insights from documents, and run generated Python code structures seamlessly.

---

## 🚀 Features

* **Multimodal Inputs:** Seamlessly process complex text prompts, files, images, and structured documents side-by-side.
* **Model Flexibility:** Easily switch between cutting-edge models depending on your task:
    * `gemini-2.5-flash` (High speed & multimodal efficiency)
    * `gemini-2.0-flash`
    * `gemini-1.5-pro` (Deep reasoning & complex data parsing)
* **Creative Precision Slider:** Real-time temperature controls ranging from precise/deterministic outputs to highly creative generations.
* **Code-Generation Hub:** Tailored to generate highly readable, accurate code blocks (such as interactive Python games and automation scripts) with structured step-by-step explanations.
* **Clean Workspace UI:** A streamlined dark-mode interface built to eliminate cognitive strain and optimize daily research and developer workflows.

---

## 🛠️ Tech Stack

* **Core AI Engine:** Google Gemini API
* **Frontend:** HTML5, CSS3 (Modern Dark-Mode UI Theme), and Vanilla JavaScript
* **Backend Support:** Python
